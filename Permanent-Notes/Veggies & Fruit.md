# Veggies

* Zucchini (Courgette, 櫛瓜/西葫蘆/夏南瓜) ↔ Gurke (Cucumber, 小黃瓜)
* 櫻桃蘿蔔 (en: Radish, de: Radieschen): 口感爽脆，帶點微辛，根、葉均可食用

# Fruit

* 鰐梨 = 酪梨 = 牛油果 = Avocado → 全球公認最好吃、油脂最豐厚的酪梨品種 - Hash (哈斯)
* 仙桃 = 蛋 (黃) 果 = Egg Fruit = Canistel
