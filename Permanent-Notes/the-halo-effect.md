= 光環效應 = 光暈效應 = 月暈效應

指人們在認知他人時，會先依據初步印象，並以此推斷對方的其他特質，形成「以偏概全」的主觀判斷，也是一種心理臆測、投射作用、刻板印象（Stereotype）、意識形態（Ideology）。

當一個人表現好時，大家對他的評價遠遠高於他實際的表現。反之，當一個人表現不好的時候，別人眼中所認為的差勁程度，也會遠大於他真正差勁的表現 [^1]。_就像我們看月亮時，看到的不僅是月亮的實際大小，還包含月亮的暈光。_

當一個人表現出色時，在他人心中的整體評價往往被大幅提升，甚至超越其實際表現。反之，若某人表現不佳，負面印象也會被放大，導致評價遠低於其真實狀況 [^1]。_這就像我們看月亮時，看到的不僅是月亮本身，還包含周圍的暈光。_

結論：第一印象的重要性！

# 應用與例子

* 在職場中，員工如果在入職初期表現出色，往往會被上司賦予更多正面特質（如責任心、領導力），即使後續表現平平，也容易獲得較高評價。
* 在學校，老師對學生的第一印象會影響後續評分與互動。例如，穿著整齊、舉止得體的學生容易被認為更聰明、更努力。
* 品牌行銷：知名品牌因為過去良好形象，消費者會自動認為其新產品也同樣優秀。
* 社交場合：外貌、談吐、氣質等初步印象會影響他人對個人能力、品格的判斷。

[^1]: 此現象稱為「反光環效應 (Reverse Halo Effect) = 魔鬼效應 (Devil Effect) = 尖角效應 (Horns Effect)」
